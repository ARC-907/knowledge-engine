# Demo Library

A minimal example showing the conventional shape of a library:

- `00-foundations/` — orientation material
- `01-applied/` — practical / applied content
- `CATALOG.md` — file inventory
- `README.md` — this file

Three-lens framework: **CONCEPT → PATTERN → APPLY** (declared per library).
