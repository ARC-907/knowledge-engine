# Demo Skill

## When to use

Pick this skill when you want to see the bare-minimum shape of a skill package.

## How it works

1. Read the inputs.
2. Apply the procedure described in `procedure.md`.
3. Return the result.

## Inputs / outputs

- **Input**: free-form text
- **Output**: same text, unchanged
