Welcome to Knowledge-Engine

You're looking at three demonstration libraries. To make this your knowledge engine:

  1. Drop your markdown into corpus/libraries/ (one folder per library).
  2. Run knowledge-engine bootstrap to register it.
  3. Run knowledge-engine reindex — or click Reindex below.
  4. Search here, from the CLI, or from your AI assistant via MCP.
