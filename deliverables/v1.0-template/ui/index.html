Welcome to Knowledge-Engine

You're looking at three demonstration libraries. To make this your knowledge engine:

  1. Drop your markdown into corpus/libraries/ (one folder per library).
  2. Run knowledge-engine bootstrap to register it.
  3. Run knowledge-engine reindex — or click Reindex below.
  4. Search here, from the CLI, or from your AI assistant via MCP.

Provider credentials

Provider-abstracted keys for upstream models (Anthropic, OpenAI, Ollama, or any custom endpoint). You bind a provider to the environment variable that holds its secret — the actual key stays in your environment and is never stored in the database. The dot shows whether that variable is currently set. Nothing ships pre-filled; the registry starts empty.

Board access keys

X-Board-Key tokens that authenticate callers to this board (only enforced when "Require key for non-localhost posts" is on). Hashed with SHA-256 — the raw key is shown once on creation, copy it before dismissing. Master-key bootstrap (via knowledge-engine board keys bootstrap-master) creates the first admin token on localhost. Starts empty.
