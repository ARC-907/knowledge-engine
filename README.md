# Knowledge-Engine

> Self-hosted, corpus-agnostic knowledge engine for your markdown notes. FTS5 search, an MCP stdio server for AI assistants (Claude Desktop, Cursor, Continue), a single-file Alpine.js + Tailwind dashboard, and an optional bge-m3 embedding index — all running locally.

This public repository contains the **free MIT-licensed engine + dashboard + demo content**. Source lives in [`deliverables/v1.0-template/`](deliverables/v1.0-template/). Clone it, point it at your own markdown corpus, run the dashboard locally. That's enough to build a useful personal or team knowledge engine.
